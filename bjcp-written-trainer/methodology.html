What This Trainer Is Teaching

Strong written exam answers are not random recipes assembled from memory. They are internally consistent beer designs built from clear technical decisions.

This trainer teaches the decision chain:

  • Estimate gravity from ABV and finishing gravity
  • Set bitterness based on style balance
  • Estimate total grist using fixed exam assumptions
  • Select mash schedules that support body and finish
  • Select fermentation profiles that match style intent
  • Compare similar styles using exam language

Trainer Assumptions

  • Metric recipe construction
  • 21.5 liters into fermenter
  • 70% brewhouse efficiency
  • ABV targets in 0.25% increments
  • Exam-speed approximation over software precision

The Gravity Shortcut

ABV × 8 = approximate gravity drop

Example: 5.0% ABV × 8 = 40 gravity points

Expected FG 1.010 → Estimated OG 1.050

Fast. Repeatable. Close enough for exam construction.

Why Balance Matters

The written exam rewards coherent beer design, not isolated memorized numbers.

A German Pils should carry firmer bitterness than a Munich Helles. A Märzen should present lower bitterness relative to gravity.

The trainer teaches bitterness as structure, not trivia.
